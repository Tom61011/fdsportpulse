# 🏋️ Fd Sport Pulse - Site Web

Site web professionnel pour l'entreprise de fitness Fd Sport Pulse.

## 📁 Structure du site

- `index.html` - Page d'accueil
- `services.html` - Page des services
- `contact.html` - Page de contact avec formulaire
- `merci.html` - Page de confirmation (optionnelle)
- `style.css` - Styles CSS
- `script.js` - JavaScript interactif
- `LOGO ENTREPRISE.png` - Logo de l'entreprise

## 🚀 Utilisation en local

1. Double-cliquez sur `index.html` pour ouvrir le site dans votre navigateur
2. Ou utilisez un serveur local (Live Server dans VS Code)

## 📧 Configuration du formulaire de contact

### Fonctionnement actuel
Le formulaire de contact utilise un système **mailto** qui ouvre directement votre client email par défaut (Outlook, Gmail, etc.). Les messages sont envoyés à : **tokvanh@gmail.com**

**Avantages :**
- ✅ Fonctionne immédiatement, aucune configuration nécessaire
- ✅ Pas besoin de service tiers
- ✅ Pas de limite d'envois

**Note :** Si vous souhaitez utiliser un système de formulaire plus avancé (comme FormSubmit ou un backend personnalisé), consultez le guide d'hébergement pour les options disponibles.

## 🌐 Hébergement recommandé

### GitHub Pages (Gratuit)
1. Créez un compte GitHub
2. Créez un nouveau repository
3. Uploadez tous les fichiers
4. Activez GitHub Pages dans les paramètres

### Netlify (Gratuit)
1. Créez un compte sur Netlify
2. Glissez-déposez le dossier du site
3. Votre site sera en ligne en quelques secondes
